Portfolio

TIELabs Academic Project: At TEIlabs, we focus on telepresence robotics. Our goal is to find ways to optimize robots to to allow teachers to teach students from remote locations. My research deals specifically with gesture space recognition. So far, I have used the OpenCV library to create an image overlaying program. Currently, I am looking into shadow removal from images using machine learning.

Remote Controlled Car Project: This is a car I made to get familiar using the Arduino uno and components such as a motor driver and a DSD TECH HM-10 Bluetooth 4.0 BLE iBeacon UART Module. The remote was created using another bluetooth module set to the master setting, a joystick and an Arduino Nano. The car is able to go in every direction with a small delay.
