Remote Controlled Car

This is a car I made to get familiar using the Arduino uno and components such as a motor driver and a DSD TECH HM-10 Bluetooth 4.0 BLE iBeacon UART Module. The remote was created using another bluetooth module set to the master setting, a joystick and an Arduino Nano. The car is able to go in every direction with a small delay.
To improve this project I plan to add a camera to the car so that I can track my dog. If the car detects my dog, it will turn in the opposite direction. I came up with this idea after my dog stepped on the car breaking its motor driver.
